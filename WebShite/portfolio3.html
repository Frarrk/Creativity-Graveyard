Portfolio 3 – Number Systems

Definition and Purpose

  • Core Concept: A number system is a system of writing designed to express numbers. It functions as a mathematical language providing a consistent method for representing numerical and abstract quantities.
  • Function: Its main purpose is to give every number a unique representation, distinguishing the quantity from the symbol used to express it.

The Decimal System (Base-10)

  • Structure: Uses a radix of 10 and digits 0–9.
  • Human Significance: The most widely used system in daily life, science, and communication.
  • Role in Computing: Computers convert binary results into decimal so humans can interpret them.

The Binary System (Base-2)

  • Structure: Uses digits 0 and 1, called bits.
  • Machine Reality: Binary is the fundamental language of all computers.
  • Physical Necessity: Computers run on transistors which only have two states: ON (1) or OFF (0), making binary the perfect fit.

The Octal System (Base-8)

  • Structure: Uses digits 0–7.
  • Historical Significance: Used as a readable shorthand for binary in older systems.
  • Application: Common in early computers like the PDP-8 due to easy grouping of bits in sets of 3.

The Hexadecimal System (Base-16)

  • Structure: Uses digits 0–9 and letters A–F, representing values 10–15.
  • Modern Standard: The main shorthand for representing binary data today.
  • Efficiency: Maps perfectly to 8-bit bytes (each byte becomes two hex digits), making it ideal for programming.

Analysis of System Roles

  • Technological Solution: Different number systems exist to solve specific hardware and software needs.
  • The Bridge:
    • Decimal: For humans
    • Binary: For hardware
    • Octal & Hexadecimal: Act as translators between humans and machines
